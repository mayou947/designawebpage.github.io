Content - HTML

The aim of design is to enhance the presentation of the content! Content is the most important element of a website.
You should spend time gathering your content! Thinking about your layout and sketching your ideas on paper.

Presentation - CSS

Styling your content will make your website more useable. For websites we use Cascading Style Sheets (CSS).
CSS is a set of rules that allow you to style content. The rules are element:style pairs, which identify the element and how it is to be styled.

Behaviour - JavaScript

JavaScript(JS) handles behaviours in a website. Most functions perfomed on the website is executed by the code in JS.
To enhance the capability of a website it is essential write great code in JS and link the document with the HTML file.
